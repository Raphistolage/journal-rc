# Armadillo-Rust Basic Integration

Simple example demonstrating Armadillo linear algebra library integration with Rust through CXX.

## Purpose

- **CXX Bridge** - Simple function calls across Rust-C++
- **Linear Algebra** - Matrix and vector operations with Armadillo from Rust
- **System Library Integration** - Using system-installed libraries from Rust
- **Data Representation** - Basic patterns for C++ data in Rust

## Requirements

- Armadillo library (`libarmadillo-dev` on Debian/Ubuntu)
- OpenBLAS and LAPACK
- C++14 compiler
